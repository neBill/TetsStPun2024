Описание

Приложение для подготовки к тестированию ЕКЭ.

Источники тестовых вопросов:

- вопросы разделов "Микротесты" - базы микротестирования экзаменационной платформы "StartExam" (где проходим микротесты и сдаем ЕКЭ). Вопросы относятся к профуровню "База";
- вопросы разделов "Охрана труда" - сайт "Конструктор тестов";
- вопросы по разрядам и профуровням (пока только газоразделение) - вопросы разработанные ИТР установки для ЕКЭ.

Основные возможности

1. Сохранение истории тестирования.

История прохождения сохраняется для каждого теста отдельно. При остановке тестирования показываются ошибки. После прохождения всего теста история сбрасывается при повторном выборе данного теста.
Есть возможность сбросить всю историю тестирования нажав пункт меню "Удалить историю". Удаляется история для всех тестов.

2. Режимы прохождения тестов.

  • "Режим контроля" - включен по умолчанию. В этом режиме результаты прохождения показываются при завершении теста или при остановке тестирования;
  • "Режим обучения" - включается в настройках. При включенном переключателе "Режим обучения", если ответ правильный, происходит автоматический переход на с ледующий вопрос. При неверном ответе - показываются верный и неверный варианты ответа. Результаты показываются при завершении теста или при остановке тестирования.

3. Опция перемешивания вопросов.

"Перемешивание вопросов" - включается в настройках. При включенном переключателе "Перемешивание вопросов" вопросы будут перемешаны при каждом выборе теста. Данный режим позволяет "погонять" вопросы в случайном порядке. При остановке тестирования показываются ошибки. В данном режиме история сохраняется только в рамках текущей сессии тестирования. При повторном выборе теста, история сбрасывается.

4. Оформление.

"Темная тема" - включает темную тему приложение. Выключение переключателя включает тему по-умолчанию;
Есть возможность изменить размер шрифта или установить значение по-умолчанию.
